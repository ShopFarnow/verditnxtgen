Zero-touch automation

Refreshes itself. Every single day.

You don't need to touch the server. GitHub Actions runs the full pipeline nightly at midnight IST.

⏰
Nightly cron job
A GitHub Actions workflow fires at 00:00 IST. It re-crawls every company in the database, recalculates scores, and pushes updated JSON — no server, no cost.
# .github/workflows/refresh.yml
on:
schedule:
- cron: '0 18 * * *'
jobs:
refresh:
runs-on: ubuntu-latest
steps:
- run: node scripts/crawl.js
- run: node scripts/score.js
